
UST Employee API - Example Outputs

Request:
GET /employees

Response:
[
  {"id": 1, "name": "Asha Rao", "email": "asha.rao@ust.com", "department": "Engineering", "role": "Engineer", "status": "active"},
  {"id": 2, "name": "Vikram S", "email": "vikram.s@ust.com", "department": "Delivery", "role": "Manager", "status": "active"},
  {"id": 3, "name": "Meera N", "email": "meera.n@ust.com", "department": "HR", "role": "HR", "status": "active"}
]

{

  "department": "Finance",
  "role": "Analyst",
  "status": "active"

  "name": "Ravi Kumar",
  "email": "ravi.kumar@ust.com",
  "department": "Finance",
  "role": "Analyst",
  "status": "active"
}

6.  Check-in Attendance

Request:
POST /employees/2/checkin
Body:
{
  "date": "2025-11-24",
  "time": "09:00:00"
}

Response:
{
  "employee_id": 2,
  "date": "2025-11-24",
  "check_in": "09:00:00",
  "check_out": null
}

7.  Check-out Attendance

Request:
POST /employees/2/checkout
Body:
{
  "date": "2025-11-24",
  "time": "17:00:00"
}

Response:
{
  "employee_id": 2,
  "date": "2025-11-24",
  "check_in": "09:00:00",
  "check_out": "17:00:00"
}

8.  Create Leave Request

Request:
POST /employees/3/leave-requests
Body:
{
  "from_date": "2025-12-01",
  "to_date": "2025-12-05",
  "reason": "Family function"
}

Response:
{
  "leave_id": 1,
  "employee_id": 3,
  "from_date": "2025-12-01",
  "to_date": "2025-12-05",
  "reason": "Family function",
  "status": "pending"
}
